# AGENTS (viewer)

## Escopo

Este diretório contém a ferramenta **Git Commits Viewer** (Tkinter).

## Regras gerais

- Manter mudanças pequenas e localizadas.
- Preservar o estilo existente do código.
- Evitar dependências novas sem justificativa.
- Preferir funções curtas e focadas.
- Comentar apenas o “porquê” quando necessário.

## Execução local

```bash
python3 tools/viewer/main.py --repo /caminho/do/repo --limit 100
```

## Observações

- O projeto roda com Python 3.12 (Tkinter).
- Ao alterar comportamento do Git, revisar efeitos colaterais e erros.
